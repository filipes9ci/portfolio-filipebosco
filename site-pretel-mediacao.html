Sobre

O projeto se iniciou com a fase de planejamento e pesquisa que, dentre outros, envolveu: análise de similares (sites de mediação e conciliação), levantamento e análise do público alvo, planejamento de teste de usabilidade com o usuário, escopo de devices e plataformas, mapeamento de requisitos técnicos para o desenvolvimento do site e gerenciamento do projeto.

Interface responsiva

Considerando a responsividade do website, o projeto se direcionou ao mobile first, onde todo o conteúdo é primeiramente organizado tendo dispositivos móveis como base (em especial smartphones) e só então é expandido para versões maiores (desktop).
